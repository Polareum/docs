# `/api/wallets`

**Method:** `GET`  

## Description
Lists wallets for the user



## Parameters
- `X-API-Key` (header): string
- `BlockchainRef` (query): string
- `NetworkRef` (query): string
- `Label` (query): string
- `Page` (query): integer
- `PageSize` (query): integer
- `OrderBy` (query): string

## Response
OK
