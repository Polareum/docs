# `/api/erc20/allowance`

**Method:** `GET`  

## Description
Get how much spender is allowed to spend owner's token



## Parameters
- `X-API-Key` (header): string
- `BlockchainRef` (query): string
- `NetworkRef` (query): string
- `TokenRef` (query): string
- `SourceWalletRef` (query): string
- `SpenderRef` (query): string

## Response
OK
