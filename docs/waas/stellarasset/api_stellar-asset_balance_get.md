# `/api/stellar-asset/balance`

**Method:** `GET`  

## Description
Returns stellar balance



## Parameters
- `X-API-Key` (header): string
- `BlockchainRef` (query): string
- `NetworkRef` (query): string
- `WalletRef` (query): string
- `Code` (query): string
- `Issuer` (query): string

## Response
OK
