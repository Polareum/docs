# `/api/polkadot/status`

**Method:** `GET`  

## Description
Returns transaction status for Polkadot networks



## Parameters
- `X-API-Key` (header): string
- `BlockchainRef` (query): string
- `NetworkRef` (query): string
- `ExtrinsicHash` (query): string
- `BlockHash` (query): string

## Response
OK
